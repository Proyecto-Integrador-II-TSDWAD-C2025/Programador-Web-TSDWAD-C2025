# Frontend - ProyectoIntegrador (Angular)

Frontend moderno construido con Angular v21+ y Bootstrap (sin CDN).

## 📦 Dependencias

- Angular 21+
- Bootstrap 5
- TypeScript 5.9+
- RxJS 7.8+

## 🚀 Inicialización Rápida

### Instalar dependencias
```bash
npm install
```

### Servidor de desarrollo
```bash
npm start
```

Navegar a `http://localhost:4200/`. La aplicación se recargará automáticamente si cambias archivos de código.

### Build para producción
```bash
npm run build
```
